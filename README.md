# config-server

| Endpoint                                    | Função             |
| ------------------------------------------- | ------------------ |
| `http://localhost:8888/application/default` | Retorna config     |
| `http://localhost:8888/actuator/health`     | Health check       |
| `POST /actuator/refresh`                    | Recarregar configs |
